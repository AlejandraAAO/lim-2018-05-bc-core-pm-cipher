Caesar Cipher

Esta web te permite encriptar tus mensajes sustituyendo cada letra por otra. Para hacerlo debes ingresar tu texto en la primera casilla, luego debes aumentar o disminuir el número de desplazamientos que corra tu letra, por ejemplo si para una "a" eliges como desplazamiento 2, te resultara la letra "c", y así sucesivamente.
