Caesar Cipher

Hola, esta web te permite encriptar tus mensajes sustituyendo cada letra por otra. Para hacerlo debes ingresar tu texto en la casilla numero 1, luego debes aumentar o disminuir el numero de desplazamientos que corra tu letra, por ejemplo si para una "a" eliges como desplazamiento 2, te resultara la letra"c", y asi sucesivamente.
